# Roadmap

## Near Term

- Device selection by input name or index
- Config file support
- Optional sensitivity presets
- Persisted calibration values
- Better logging around action failures

## Cross-Platform

- Linux shortcut backend
- Windows shortcut backend
- Platform-specific install notes

## Detection Improvements

- adaptive noise floor
- short band-pass prefilter
- more explicit whistle envelope scoring
- optional different sounds for copy and paste

## UX Improvements

- menu bar or tray mode
- startup banner with active settings
- dry-run mode for tuning without sending shortcuts
- optional sound feedback after actions
